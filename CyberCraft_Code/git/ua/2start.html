Налаштувати Git

Тепер дайте Git знати, хто ви. Це важливо для систем контролю версій, оскільки кожен коміт Git використовує цю інформацію:

git config --global user.name "test"
git config --global user.email "usermail@gmail.com"

git - команда до якої звертаємося

--global - щоб установити ім’я користувача та електронну адресу для кожного репозиторію на вашому комп’ютері. Якщо ви хочете встановити ім’я користувача/електронну адресу лише для поточного репозиторію, ви можете видалити global

mkdir myproject
cd myproject

mkdir - створює новий каталог (папку чи теку. Як вам зручніше)

cd - переходить до цієї папки. Якщо простими словами, то переходить до місця цієї папки.
Щоб ініціалізувати Git - треба виконати команду:

git init
Initialized empty Git repository in /Users/user/myproject/.git/
І ви щойно створили свій перший репозиторій Git. P.s. Git тепер знає, що він має стежити за папкою, яку ви ініціював це. Git створює приховану папку для відстеження змін.

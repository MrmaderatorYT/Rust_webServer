Додавання нових файлів

Умовно у нас є файл data.txt. В ньому якісь дані (умовно). Тепер зайдемо до терміналу й пропишемо команду

ls
data.txt
ls - виведе список файлів у каталозі. Ми побачимо, що це data.txt існує.
git status

On branch master

No commits yet

Untracked files:
(use "git add ..." to include in what will be committed)
data.txt

nothing added to commit but untracked files present (use "git add" to track)

Тепер Git знає про файл, але не додав його до нашого репозиторію! Файли у вашій папці сховища Git можуть перебувати в одному з 2 станів:
  • Відстежувані – файли, про які Git знає та які додаються до репозиторію
  • Untracked - файли, які знаходяться у вашому робочому каталозі, але не додані до сховища
Коли ви вперше додаєте файли до порожнього сховища, усі вони не відстежуються. Щоб змусити Git відстежувати їх, вам потрібно створити їх або додати до середовища розміщення.

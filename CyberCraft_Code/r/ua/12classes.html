Класи

В мові програмування R можна створити класи. В R існують два основних типи систем класів: S3 та S4

S3 клас

S3 класи - це найбільш простий та найпопулярніший тип класів у R. Вони дозволяють нам легко визначити класи та методи для цих класів без великої кількості коду. Як створюється клас такого типу:

setClass("Person", slots = c(name = "character", age = "numeric))

В цьому прикладі ми створюємо клас Person з двома атрибутами. Також ми сказали, що атрибут name має рядковий тип даних, а age - числовий.
Тепер створимо об'єкт:

person1 <- new("Person", name = "Abra", age = 30)
Ми створили змінну, яка є об'єктом класу Person, потім ми записали атрибути. Створюємо об'єкт за допомогою функції new.
Тепер, ми можемо додати метод до класу, тобто створюємо методи для роботи з цим класом:
print.Person <- function(x){
cat("Person:", x@name, ", Age:", x@age, "\n")
}
Тобто цей метод буде використовуватися для друку об'єктів класу Person. Також, тут присутній символ @. Він означає, що ми отримуємо доступ до слотів(по іншому атрибути, але змінна slots в параметрах класу говорить сама за себе) об'єкта використовується цей символ. Тобто, коли ми хочемо отримати доступ до слоту (атрибуту) об'єкта, ми використовуємо спочатку ім'я об'єкта, а потім ім'я слоту. В нашому випадку об'єкт x слоту(атрибуту) name та age.
Тепер використаємо об'єкт:
print.Person(person1)
#Person: Abra , Age: 30

S4 класи

Для роботи з цим типом класів ми повинні використовувати бібліотеку methods:
library(methods)
setClass("Car", slots = c( make = "character", model = "character", year = "integer" ))
У цьому прикладі ми визначаємо клас Car з трьома атрибутами: make (марка автомобіля), model (модель автомобіля) і year (рік випуску).
Тепер створимо об'єкт цього класу:
my_car <- new("Car", make = "Toyota", model = "Camry", year = 2020)
Тепер додамо методи до класу. Для цього використовується функція setMethod:
setMethod("show", "Car", function(object) {
cat("Car Make:", object@make, "\n")
cat("Car Model:", object@model, "\n")
cat("Year:", object@year, "\n")
})
Цей метод show буде використовуватися для виведення деталей про автомобіль при виклику функції print або просто введення об'єкта my_car в консолі.
Тепер використаємо об'єкт:
print(my_car)
Car Make: Toyota
Car Model: Camry
Year: 2020

Вектори

Вектори - це основний тип даних, який представляє собою одномірний масив об'єктів одного типу. Вектори можуть бути числовими, символьними, логічними та іншими.
Ось приклад створення вектора:

numeric_vector <- c(1, 2, 3, 4, 5)
char_vector <- c("apple", "banana", "peach", "cherry")
bool_vector <- c(TRUE, TRUE, FALSE, TRUE, FALSE)

В даному випадку ми створили вектори. Щоб створити вектор треба використати функцію c. Далі через кому ми записуємо дані. Дані повинні бути одного типу
. Щоб отримати дані з вектора ми використовуємо індекси:

print(numeric_vector[1]) print(char_vector[2]) print(bool_vector[4])

Зверніть увагу

Індексація векторів починається з одиниці. Тобто в попередньому прикладі ми отримає значення:
1
banana
TRUE

Щоб змінити значення окремого індексу ми звертаємося до вектора і його ж індекса:

numeric_vector[1] <- 99
char_vector[3] <- "pear"
Ми змінили перший елемент вектора numeric_vector з 1 на 99. Також, ми змінили 3 елмент вектора char_vector на pear
Найцікавіше - з векторами можна виконувати арифметичні операції:
x <- c(1, 2, 3)
y <- c(4, 5, 6)
sum <- x + y
#5 7 9

Зверніть увагу

Кількість елементів обох векторів повинна бути однакова.

Функції для роботи з векторами

  • length(): Повертає кількість елементів в векторі.
  • mean(): Обчислює середнє значення.
  • median(): Обчислює медіану.
  • max() і min(): Знаходять максимальне і мінімальне значення вектора.
values <- c(5, 10, 15, 20)
print(length(values)) # Виведе 4
print(mean(values)) # Виведе 12.5
print(max(values)) # Виведе 20

Логічні вектори

Логічні вектори загалом використовуються для фільтрації даних:
values <- c(1, 2, 3, 4, 5)
logical_vector <- numbers > 3
print(logical_vector) # Виведе TRUE FALSE FALSE TRUE TRUE
print(numbers[logical_vector]) # Виведе 4 5
Розбираємо:
У першому рядку ми створюємо вектор
У другому рядку ми створюємо логічний вектор logical_vector. Кожен елемент цього вектора буде TRUE, якщо відповідний елемент вектора numbers більше 3, і FALSE в іншому випадку. Таким чином, logical_vector буде виглядати так: FALSE, FALSE, FALSE, TRUE, TRUE.
У третьому рядку ми виводимо логічний вектор logical_vector. Вивід буде: TRUE FALSE FALSE TRUE TRUE, що відповідає нашому вектору logical_vector.
У четвертому рядку ми ми використовуємо logical_vector для індексації вектора numbers. Коли ви використовуєте логічний вектор як індекс, R виводить лише ті значення, для яких відповідні значення в логічному векторі є TRUE. Таким чином, numbers[logical_vector] повертає підмножину вектора numbers, яка містить лише ті елементи, які більше 3. Отже, результатом буде виведення чисел 4 та 5.

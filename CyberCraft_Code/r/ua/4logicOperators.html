Порівнювальні та логічні оператори

Порівнювальні оператори

Порівнювальні оператори повертають логічне значення, а саме TRUE / FALSE

  • Рівність: '=='
    x <- 3==3
    #TRUE
  • Нерівність: '!='
    x <- 3!=6
    #TRUE
  • Більше: '>'
    x <- 5L>2L
    #TRUE
  • Менше: '<'
    x <- 5<2
    #FALSE
  • Більше або дорівнює: '>='
    x <- 2>=2
    #TRUE
  • Мешне або дорівнює: '<='
    x <- 10<=3
    #FALSE

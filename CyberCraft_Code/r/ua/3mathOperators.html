Математичні оператори

Існує 7 арифметичних операцій в мові програмування R

  • Додавання: '+'
    x <- 3+3
    #6 До речі, це коментар в мові R)
  • Віднімання: '-'
    x <- 3-6
    #-3
  • Множення: '*'
    x <- 5L*2L
    #10 (Без L)
  • Ділення: '/'
    x <- 5/2
    #2.5
  • Піднесення до степеня: '^'
    x <- 2^5
    #32
  • Ділення по модулю (залишок від ділення): '%%'
    x <- 10%%3
    #1
  • Цілочисельне ділення: '%/%'
    x <- 10%/%3
    #3

NaN, NULL, Inf

NaN, NULL, Inf - три спеціальні значення

NaN (Not a Number)

  • Поведінка у векторах: NaN використовується для позначення випадків, коли значення не є числом, таким як результат ділення на нуль або математичні операції, які не мають сенсу (наприклад, sqrt(-1))
    P.s. Вітання від комплексних чисел.
  • Поведінка при перевірці: is.nan() визначає, чи є значення NaN.
  • Використання: NaN використовується для позначення відсутності валідного числового значення.

NULL

  • Поведінка у векторах: NULL використовується для позначення відсутності об'єкта.
  • Поведінка при перевірці: is.null() визначає, чи є значення NULL.
  • Використання: NULL використовується для позначення відсутності об'єкта чи для встановлення змінної на порожній об'єкт.

Inf (infinity)

  • Поведінка у векторах: Inf використовується для позначення позитивної нескінченності. -Inf використовується для позначення негативної нескінченності.
  • Поведінка при перевірці: is.infinite() визначає, чи є значення Inf або -Inf.
  • Використання: Inf використовується як результат ділення числа на нуль або при інших математичних операціях, які породжують нескінченності.

Використання цих значень у векторах

my_vector <- c(1, 2, NaN, 4, NULL, Inf)
У цьому векторі NaN та Inf поведуться як звичайні числові значення, але NULL не є дійсним числовим значенням і вимагає особливої обробки.

Використання (коротко)

NaN: При математичних операціях, які не можна виконати, але потрібно повернути результат.
NULL: Коли вам потрібно позначити відсутність об'єкта.
Inf: Коли результат математичної операції є нескінченним.

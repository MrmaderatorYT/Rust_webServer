Списки

Списки в R є складовим типом даних, який може містити різні типи об'єктів, включаючи числа, символи, інші списки, функції та навіть NULL.


Створюємо список:
my_list <- list(name = "Abra", age = 30, scores = c(85, 90, 78))

В даному випадку ми створюємо змінну my_list, яка містить три елменти - ім'я, вік, вектор з 3ма значеннями.
Також, щоб отримати дані зі списку, ми звертаємося за індексами:

my_list <- list(name = "Abra", age = 30, scores = c(85, 90, 78))
print(my_list[1])
#Abra

Звернення по імені

my_list <- list(name = "Abra", age = 30, scores = c(85, 90, 78))
print(my_list$scores)
#85 90 78
Таким чином можна звернутися до вектора:
my_list <- list(name = "Abra", age = 30, scores = c(85, 90, 78))
print(my_list$scores[2])
#90
Можна також створити список в списку:
my_list <- list(name = "Abra", age = 30, scores = list(a = 85, b = 90, c = 78))
print(my_list$scores$c)
#78
Щоб змінити дані елемента в списку ми можемо так само працювати, як і із векторами:
my_list <- list(name = "Abra", age = 30, scores = list(a = 85, b = 90, c = 78))
my_list[1] <- "John"
#Або ж можна звертатися по імені:
 my_list$name <- "John"

Функції в списку

:Списки можуть містити функції:
my_list <- list(name = "Abra", age = 30, scores = list(a = 85, b = 90, c = 78))
my_function <- function(x){
return(x^2)
}
my_list$func <- my_function
Зараз у списку my_list є новий елемент func, який є функцією my_function.

Пару питань

Символ долару '$' в R використовується для отримання доступу до елементів списку за їх назвою або ідентифікатором. Він дозволяє звертатися до конкретних елементів списку без необхідності використовувати індексацію.
Зауважте, що використання $ працює лише зі списками та фреймами даних. Якщо ви намагаєтеся використовувати $ з іншими типами даних, ви отримаєте помилку.
Також, якщо ім'я елемента списку містить пробіл або спеціальний символ, ви повинні використовувати двокрапку та подвійні квадратні дужки для звернення до нього:
my_list <- list("first name" = "John", age = 30)
print(my_list$"first name") # Виведе "John"

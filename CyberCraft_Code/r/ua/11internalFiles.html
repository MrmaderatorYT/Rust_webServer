Взаємодія з іншими .R файлами

Умовно у нас є 2 файли: script.r - тут буде вся логіка. Інший файл - main.r - головний файл, де буде відбуватися звернення до файлів та виконання логіки з інших файлів.
Опишимо файл script.r:

#script.r
x <- c(1, 2, 3, 4, 5)
mean_value <- mean(x) #середнє арифметичне значення
print(mean_value) #Вивід цього середнього значення

Тепер опишемо файл main.r:

source("~/script.r")
Символ ~ - вказує на те, що файл script.r знаходиться в тій же папці, що й файл main.r, далі ставимо символ слешу і повну назву файлу.
Якщо файл знаходиться в іншому місці - вказуємо повне розсташування файлу.
Запустивши файл - ми побачимо середнє арифметичне.
Але, що якщо нам потрібно виконати певну дію в певному місці?
Для цього в нас є функції. Перепишимо файл script.r:
#script.r
my_function1 <- function() {
# код для першої функції
print("Function 1 executed!")
}
my_function2 <- function() {
# код для другої функції
print("Function 2 executed!")
}
Тепер змінимо файл main.r:
#main.r source("~/script.r")
my_function1()
my_function2()
І тепер отримаємо те, що записано у функціях.

Умовні конструкції

Умовні конструкції - дозволяють виконувати певний блок коду залежно від умови. Ось як вони працюють:

x <- 10
if (x > 5){
print("x is higher than 5")

В даному випадку ми отрмаємо в консолі x is higher than 5 так як х більше за 5.
Розглянемо такий випадок:

x <- 4
if (x > 5){
print("x is higher than 5")

В даному випадку ми нічого не отримали так як ми лише опрацювали момент, коли змінна х більше за 5, але не опрацювали момент, коли x менше або дорівнює за 5. Для цього в нас є блок else. Виглядає це так:

x <- 4
if (x > 5){
print("x is higher than 5")
}else{
print("x is less than 5")
}
В даному випадку ми отрмаємо вивід x is less than 5.

Зверніть увагу

if (умова){
якась дія
}
else{
Інша дія
}
В мові R ми не можемо поставити блок else з нового рядка, там де немає іншої, закриваючої фігурної дужки. Тобто в даному прикладі ми отримаємо помилку компіляції. Отже єдий вихід такий:
if (умова){
якась дія
}else{
Інша дія
}
Також, якщо вам потрібно зробити декілька перевірок, то існує else if. Виглядає це так:
x <- 5
if (x > 5){
print("x is higher than 5")
}else if (x==5){
print("x equals 5")
}else{
print("x is lower than 5")
В даному випадку ми отрмаємо в консолі текст x equals 5

Логічні оператори

  • Логчне або: '|'. Цей оператор шукає хоча б одну умову, яка дорівнює значенню TRUE:
    x <- 4
    if (x==3 | x >= 4){
    print("Yeah"
    }
    #YEAH
    • Також є інший варіант - ||. Цей тип відрізняється від першого лише тим, що він шукає найперше значення, яке дорівнює TRUE. Якщо знайшов - далі не перевіряє, а виконує блок виконання коду.
  • І (та / й): '&'. Умовно у нас є такий код:
    x <- 4
    if (x==3 & x >= 6){
    print("Yeah")
    }else{
    print("Not yeah")
    • Отже. Цей опертор логічного "і" перевіряє, щоб всі умови мали значення "TRUE", а якщо хоча б одне має FALSE - код в блоці не буде виконуватися.
    '&&' - так само, але шукає, щоб хоча б одна з умов мала значення FALSE, то зупиняється далі йдуть інші умовні блоки, якщо вони є.
  • Ні: '!'. Робить навпаки (умовно):
    x <- 1
    if (x != 1){
    print("x is not 1")
    }
    В даному випадку ми нічого не отримаємо, бо х дорівнює 1, але тепер додамо '!':
    x <- 1
    if !(x != 1){
    print("x is not 1")
    }
    Тепер ми отримаємо в консолі x is not 1. Так як '!' робить заперечення на перевірку.

Типи даних

В мові програмування R існує 11 типів даних:

  • Numeric: цей тип даних включає в себе всі дійсні числа. Наприклад: 3.14, 42, -0.001.
    x <- 3.14
    x1 <- -3246
    x3 <- -23.4546
  • Integer: Цілі числа без десяткової частини. Наприклад: 5L (де L вказує на ціле число).
    x <- 5L
    x1 <- -234L
    x3 <- 2546L
  • Character: Цей тип даних використовується для зберігання тексту. Рядки завжди заключені в подвійні (" ") або одинарні (' ') лапки.
    x <- "Hello world"
    x1 <- 'Hello world!'
     x3 <- "'Hello world in another world!'"
    x4 <- "Hello world 'in' another world!"
    Якщо вам потрібні лапки в якості символа,то в R існує можливість створювати текстові змінні двома шляхами - з використанням одинарних і подвійних лапок. Наприклад, змінна х2 буде мати текст 'Hello world in another world', а змінна х3 - Hello world 'in' another world, тобто, якщо вам потрібні символи лапок - використовуйте таким чином.
  • Logical: Може приймати значення TRUE або FALSE. Цей тип даних часто використовується для умовних перевірок та булевої логіки.
    x <- TRUE
    x1 <- FALSE
    Зверніть увагу, слова: TRUE, FALSE пишуться усі літери з великої букви.

Це є найосновнішими типами даними. Є ще вектори, списки, матриці, масиви, фактори, дата й час, але це згодом

x <- 1

В даному випадку ми створюємо змінну х, значення якої дорівнює 1

Коментарі

Коментарі - частина коду, який не сприймає компілятор, ігноруючи його. Коментарі використовують зазвичай для опису коду, позначок від розробників.


Існує 2 види коментарів
  • Олнорядковий
  • Багаторядковий
#Однорядковий коментар
AGE = 45
echo "Age: $AGE" #вивід тексту

Тут ми використали одинарний коментар. Одинарний коментар позначається символом решіткою (хештегом): #. Після цього символу ми можемо писати текст, який компілятор ьуде ігнорувати. Також слід звернути увагу на те, що якщо перед символом # є якийсь текст - цей текст не буде вважатися коментарем. Тобто в нашому випадку у нас виведеться в консолі текст: Age: 45

: '
multiline comments example1
multiline comments example2
multiline comments example3
'
 echo "testing multi-line comments"
Цей синтаксис включає:
  • Коментарі починаються з двокрапки :, а потім '
  • Після цього було кілька рядків коментарів
  • Коментарі закінчуються '

Це корисно для включення більшої кількості тексту, що охоплює кілька рядків, а також для документації.

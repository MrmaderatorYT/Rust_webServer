Переваги Rust

  • Легкий в навчанні
  • Об'єктно-орієнтована мова програмування
  • Альтернатива С/С++
  • Наявність користувацьких бібліотек з відкритим кодом

Структура програми

fn main()
{
println!("Hello Rust!"); //виводимо текст на екран
}

fn main() - main - слово, яке вказує, що це стартова точка програми. Без неї не може запуститись програми.
{ - початок функції.
} - кінець функції.

В Rust є коментарі. Комантарі ігнорється копілятором і використовуються як позначки для людей. Їх два види:
//абра Це однорядковий коментар. Він робить тільки один рядок не виконуваним
абра123
...

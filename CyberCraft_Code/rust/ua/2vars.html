Змінні

Для зберігання даних у програмі у мові Rust використовуються змінні. Кожна змінна характеризується трьома елементами: назвою змінної, типом даних змінної та її значенням. Для визначення змінної Rust застосовується ключове слово let
let назва змінної: тип змінної;

Після ключового слова let слідує назва змінної. Потім через двокрапку вказується тип змінної. Оскільки визначення змінної представляє окрему інструкцію, воно закачується точкою з комою.

let age: u32;

В данному випадку у нас створюється змінна age, яка має тип змінної u32
u32 - тип даних, які мають цілі числа, які є позитивні.

Після визначення змінної їй можна надати значення. Присвоюване змінної значення має відповідати її типу. Наприклад, вище змінна age має тип u32, тобто ціле число. Тому їй можна присвоїти лише ціле число. Для надання змінної значення застосовується знак

let age: u32;
age = 36;

В даному випадку, змінна age має значення 36. Коли ми вказали значення змінній age - ця змінна стала ініціалізованою, тобто змінною, яка має значення. В першому рядку, де ми просто створили змінну - вона є неініціалізованою.
Також можна відразу привласнити змінну значення при її визначенні:

let age: u32 = 36; Змінна в Rust обов'язково повинна мати певний тип. Однак якщо при оголошенні змінної відразу ж привласнюємо їй значення, то можна тип не вказувати. А Rust сам виходячи з наданого значення виведе тип змінної: let age = 32;
По стандарту тип даних для цілих чисел (як і в цьому приладі) в Rust є i32.
Після того, як ми визначили та ініціалізували змінну, ми можемо її використовувати. Наприклад, виведемо значення змінної на консоль:
fn main(){
let age: u32;
age = 36;
println!("Age = {}", age);
}
Для виведення на консоль застосовується макрос println!. Він виводить рядок тексту, який полягає в подвійні лапки:
println!("Age = ");
А щоб додати до рядка значення змінної, у рядку застосовуються фігурні дужки "{}". А змінна змінна вказується після рядка через кому:
println!("Age = {}", age);
Тобто замість фігурних дужок вставлятиметься значення змінної age.
Подібним чином ми можемо виводити на консоль значення відразу кількох змінних:
fn main(){
let age = 36;
let name = "Tom";
println!("Name = {} Age = {}", name, age);
}

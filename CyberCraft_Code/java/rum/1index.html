Introducere în Java

De ce ar trebui să înveți Java?


  1. Java nu își va pierde în curând relevanța
  2. Java se dezvoltă activ
  3. Abilitatea de a scrie cod pentru diverse dispozitive (PC, telefon, ceas inteligent, smart TV)
  4. Sintaxă similară pentru scrierea codului pentru diferite platforme

Avantajele Java

  • Usor de invatat
  • Limbajul de programare orientat pe obiecte
  • Se schimbă rar
  • Disponibilitatea bibliotecilor personalizate open source
  • Este posibilă migrarea la un nou limbaj de programare C, C++, C#

Compilator- un program de calculator este un program care convertește codul sursă scris într-un anumit limbaj de programare în cod echivalent care este de obicei necesar pentru a executa programul de către o mașină, de exemplu, un computer

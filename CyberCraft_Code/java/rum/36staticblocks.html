Blocuri statice

Blocurile statice sunt folosite pentru a inițializa static o clasă. Acest cod din interiorul blocului static este executat o singură dată: când class este încărcată în memorie pentru prima dată și arată astfel:

static{
System.out.println(„Bloc static”);
}

De asemenea, blocul static va funcționa fără metoda Main dacă utilizați JDK versiunea 1.6 și ulterioară (1.5, 1.4 etc.):

class Main{
static{
System.out.println(Funcționează! :0");


În caz contrar, adică pentru versiunea 1.7 și versiunile ulterioare, vom primi o eroare. De asemenea, aceste metode nu au nume și nici modificator de acces, așa că nu pot fi apelate și vor fi executate doar în această clasă. De asemenea, cu emulare, blocurile statice nu vor fi executate la pornirea programului.

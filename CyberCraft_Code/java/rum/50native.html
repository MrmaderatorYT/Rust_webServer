Aceasta este o metodă a clasei Object
native - Java Native Interface (JNI) - un mecanism pentru rularea codului sub controlul Java Virtual Machine (JVM), care este scris în limbaje de programare precum C sau C++ sau Assembler și compilat sub formă de biblioteci dinamice (cum ar fi .dll - Windows, .so - Android) permite să nu se folosească căscatul de comunicare statică Cu cuvinte simple, acesta este un lucru care vă va ajuta să lucrați cu biblioteci în diferite limbi. Pentru a-l descărca, trebuie să introduceți codul

System.loadLibrary("NativeLib");

Din păcate, nu vor exista exemple grozave, deoarece metodele native aproape că nu există, pentru că sunt foarte rare, dar foarte puternice. Putem lucra cu sisteme de design grafic precum OpenGL și așa mai departe.

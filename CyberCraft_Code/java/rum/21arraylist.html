Matrice dinamică (ArrayList)

ArrayList - matrice dinamică, își poate schimba lungimea. Când această matrice internă este plină, ArrayList creează o nouă matrice, copiază datele din cea veche, scrie în cea nouă și mărește dimensiunea. Iată formula pentru dimensiunea noii matrice:
Dimensiunea noii matrice = (dimensiunea vechii matrice * 1,5)+1
După scrierea datelor într-o nouă matrice, cea veche este șters. Iată un exemplu standard:

ArrayList list = new ArrayList();

Am creat o matrice dinamică cu o nouă listă. Spre deosebire de alte matrice, nu specificăm tipul de date aici, deoarece matricele dinamice pot accepta orice tip de date, chiar și date dintr-o altă clasă. Și aici trecem la generice:
Generice sunt tipuri cu un parametru. În Java, claselor de containere li se permite să-și specifice obiectele interne. Când declarăm o variabilă generică, specificăm nu un tip, ci două:

  1. tip variabil
  2. tip de date

ArrayList<String>
ArrayList<Integer>

După cum puteți vedea, am specificat tipul de date între paranteze drepte, cunoscute ca simboluri mai mari decât și mai mici decât. De ce nu am pus pur și simplu un tip de date normal precum int și altele între paranteze? Deoarece, ne referim la clasa wrapper a tipului. Iată un exemplu:

ArrayList <String> list = new ArrayList<>();
list.add("Hi");

Am creat o matrice dinamică de tip String named list. Pe linia următoare, am folosit metoda add, care adaugă o valoare matricei. Dar adaugă acest sens la sfârșit. Există și alte metode pentru ArrayList.
  • add(index, element) - această opțiune înregistrează valoarea elementului specificat. Valoarea indexului este numărul de ordine pentru înregistrare. Dacă nu este specificat niciun index, un element va fi adăugat la sfârșit.
  • get() - returnează valoarea elementului specificat.
  • contains() - verifică dacă elementul specificat este conținut în matrice.
  • remove() - elimină un element din matrice.
  • clear() - șterge matricea, face ca dimensiunea acesteia să fie zero.
  • indexOf() - returnează indexul elementului căutat.

Modele de argumente în metodele generalizate

Să presupunem că avem o metodă care acceptă date de un anumit tip:

public int method(){
}

Dar nimeni nu ne-a interzis să specificăm un tip necunoscut. Modelul de argument este specificat de ? și este un tip necunoscut:

boolean method(Class<?> name){
//descriere
}

De asemenea, putem limita gama de obiecte prin specificarea unei superclase:

Metoda public void (ArrayList<? extends SuperClss> variable){
//descriere
}

În acest caz, puteți folosi clase care sunt succesoare ale SuperClss și nu putem folosi String și altele dacă nu au fost folosite în SuperClss

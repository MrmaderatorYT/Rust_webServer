for-each cucle

For-each cucle - o variație a buclei for, care este utilizată atunci când trebuie să procesați toate elementele o matrice sau altceva. Chiar și traducerea acestui ciclu (pentru fiecare) înseamnă din engleză - pentru fiecare. Dar cuvântul fiecare nu este folosit. Iată cel mai simplu mod de a scrie pentru fiecare:

for(variabilă de tip: listă){
//code
}

type - tipul variabilei, care trebuie să aibă același tip ca și elementele matricei.

variabilă - numele variabilei.

list - o listă de date care trebuie procesate.
Acesta a fost un exemplu de creare a unei noi variabile cu propriul tip. Iată o variație cu o variabilă existentă care a fost creată înainte de buclă:

int var;
cucle(var: list){
//code
}

Acest cod poate fi scris cu o cucle obișnuită for:

int var;
for(var i < list.length; var++){
//code
}

Dar atunci când utilizați cucle for-each, aveți grijă la unele limitări, deoarece această buclă este mai puțin versatilă decât bucla obișnuită for

  • Dacă doriți să parcurgeți lista de la sfârșit, puteți utiliza cucle for astfel:
    for(int i = list.length-1; i>= 0;i--){
    System.out.println(lista[i]);
    }
    Dar folosind pentru-fiecare este imposibil să faci asta.

  • cucle for-each nu este potrivită dacă trebuie să faceți modificări la matrice, cum ar fi adăugarea unui element sau modificarea dimensiunii matricei

Ключове слово throws

Ключове слово throws використовується в сиґнатурі методу чи класу і вказує, що він викидує виключення. Його можна використовувати, щоб передавати виключення по стеку викликів і вказувати, що ці виключення не обов'язково повинні оброблюватися в методі в якому вони викликані. Зазвичай його використовують при роботі з файлами:

class Main throws IOException

Це ключове слово виглядає так з методами:

void method() throws Exception

Introducerea datelor în consolă. Clasa de Scanner

De ce am selectat o clasă Scanner pentru o lecție întreagă?
Scanner - o clasă care va ajuta la introducerea datelor în consolă.

Початок роботи

Pentru a începe să lucrați cu această clasă, trebuie să o importați mai întâi:

import java.util.Scanner;

importul are loc datorită cuvântului cheie import
Importul este scris înainte de declarația clasei:

import java.util.Scanner;
public class Main{
public static void main(String[] args){
}
}

Acum să creăm o variabilă de tip Scanner:

Scanner scanner = new Scanner(System.in);

Scanner - clasa importată, este ca un tip de variabilă, cum ar fi int, dar din clasa Scanner.
apoi am creat numele scanner numele poate fie orice.< br>Punem simbolul „=” pentru a atribui unele date.
new este necesar pentru a crea un nou obiect bazat pe clasa Scanner
Am rescris Scanner pentru a face referire la metoda clasei Scanner.
System.in este un parametru care va permite citirea datelor din consolă, dar acest lucru nu este suficient.
Trebuie să creăm o variabilă:

String s = scanner.nextLine();

După cum puteți vedea, am creat o variabilă s de tip String și am atribuit valoarea variabilei de scanare pe care am creat-o recent.
.nextLine() - se întoarce la date, găsește acolo următorul rând, pe care nu a citit-o încă și îl returnează.
Există, de asemenea, diferite metode de tip next:

  • .nextInt - citește și returnează numărul introdus.

    • In schimb nextInt puteți specifica un alt tip ca .nextByte si asa mai departe

  • .hasNextInt - metoda verifică dacă următoarea porțiune de intrare este un număr sau nu (returnează true sau, respectiv, false

    • În loc de hasNextInt, puteți specifica un alt tip, cum ar fi .hasNextByte si asa mai departe

assertți

Assert este un construct special care vă permite să verificați ipotezele despre valoarea datelor arbitrare într-un loc arbitrar din program.
Acest design poate semnala automat când sunt detectate date incorecte, ceea ce duce de obicei la o întrerupere de urgență a programului. Dar, la prima vedere, acesta este un design ciudat, care poate „strânge” programul într-un moment inoportun. Și apare întrebarea „De ce avem nevoie de ea?” Acest design va ajuta if la un moment dat unele date ale programului au devenit incorecte și nu am „crash” programul imediat, ci am continuat lucrul ca și cum nimic nu s-ar fi întâmplat. Deoarece assert încheie programul imediat după detectarea datelor incorecte, vă permite să localizați și să remediați rapid erorile din program care au condus la date incorecte. Acesta este scopul său Main.
Assert vă permite să detectați erori în program în timpul compilării sau în timpul execuției. Sintaxa construcției este simplă:

assert condiție;

O condiție este o expresie care, atunci când este calculată, ar trebui să dea un rezultat boolean. Și if rezultatul este adevărat, atunci assertția este adevărată și nu se ia nicio măsură.
if condiția returnează false, înseamnă că a avut loc o eroare și că obiectul de excepție implicit este creat: AssertionError
Există o altă formă de scriere a assert:

assert condiție: valoare;

Valoarea este ceea ce este transmis constructorului AssertionError. Această valoare este convertită într-un șir și afișată if condiția este falsă. Iată un exemplu de utilizare a assert

Modificatori de acces

Există un total de 4 tipuri de modificatori de acces

  • private - clasă privată sau membru al clasei. O clasă închisă sau un membru al clasei este disponibil numai pentru clasa în care este scris.
  • public - public, clasă publică sau membru al clasei. Complet opusul private. Câmpurile și metodele declarate cu acest modificator sunt disponibile pentru toate celelalte clase.
  • protected - această clasă sau membru al clasei este accesibilă de oriunde în clasa curentă și în alte clase care l-au moștenit
  • default - acest modificator nu este scris. Este standard. Acest modificator este folosit atunci când obiectul sau clasele nu îndeplinesc funcții speciale.

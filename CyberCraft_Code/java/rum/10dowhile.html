Bucla do while

Bucla do-while - foarte similar cu o buclă normală while.
Execuția acestei bucle începe cu cuvântul cheie do, urmat de comenzile de executat pentru fiecare buclă între paranteze . Acesta este urmat de cuvântul cheie while, iar între paranteze scriem condiția care este verificată de fiecare dată. Iată imaginea standard a buclei:

do{
echipe
}
while(condiție);

Dacă în cuvinte simple, această buclă face mai întâi, adică execută codul și apoi verifică. O buclă normală while verifică mai întâi și apoi execută codul

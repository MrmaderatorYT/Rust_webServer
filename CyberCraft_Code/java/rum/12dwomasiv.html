Matrice bidimensională

Crearea unei matrice bidimensionale este foarte asemănătoare cu scrierea unei matrice unidimensionale:

tip [ ] [ ] nume = new tastați [dimensiune] [dimensiune];

După cum puteți vedea, am scris patru paranteze drepte după tip. Noi suntem cei care am creat expunerea la matricea bidimensională. O matrice bidimensională este un tabel.
Primele paranteze pătrate sunt lungimea matricei, dar lungimea șirului

Al doilea paranteze drepte sunt coloane.

Să creăm un exemplu:

int [ ] [ ] numbers = new int [3] [5];

Această comandă poate fi scrisă în două rânduri:

int [ ] [ ] numbers;
numbers = new int [3] [5];

Deci, comanda int și patru paranteze împreună cu variabila numere declară variabila numere a unui tablou bidimensional. Ca rezultat al executării comenzii

numbers = new int [3] [5];

Este creată o matrice bidimensională de numere întregi: trei rânduri și cinci coloane, astfel încât această matrice poate conține 15 elemente. Pentru a obține un element dintr-o matrice de acest tip, trebuie să scrieți următorul cod:

System.out.println(numbers[1] [2]);

Vom obține numărul din al doilea element al rândului și care se află în a treia coloană. Și acum să trecem la inițializarea matricei bidimensionale:

char [ ] [ ] s = {{'A', 'B', 'C'},{'D', 'E', 'F'}};
numbers = new int [3] [5];

Aici am declarat o matrice de tipul char și am scris valorile acolo. Există, de asemenea, anumite acțiuni cu matrice:
.lenght - lungimea matricei. Avem în vedere un șir, adică pentru o matrice 3*3 lungimea este 3, iar pentru o matrice 3*4 lungimea este 3.
.clone - această comandă alocă un nou spațiu de memorie și îi atribuie matricea copiată. Acestea sunt anumite echipe, dar nu toate. Este suficient pentru moment.

Bucla for

Bucla for nu este aceeași cu bucla while, dar este mai convenabil de utilizat. Diferența dintre bucla while este că bucla for trebuie să fie executată de un anumit număr de ori când se cunoaște numărul de interacțiuni . Interacțiunea este aplicarea repetată a unei operații matematice. Șablonul standard al acestui ciclu este următorul:

for (initializare; condiție; expresie matematică){
echipe
}

În blocul de inițializare creăm o variabilă nouă sau scriem una existentă.
În blocul de condiții verificăm condiția pe care o implementăm în acest bloc. Aici vom avea sintaxa pentru scrierea blocului if, unde vom scrie condiția care este verificată.
În blocul expresii matematice scriem... o expresie matematică. Aceasta este de obicei o creștere sau descreștere.
Iată o vedere simplă a unei bucle for.

for(int k = 1; k < 5){
System.out.println(k);
}

Vom obține patru valori în consolă: 1,2,3,4. După cum putem vedea, primul bloc este executat primul, adică declarația variabilei k. Al doilea bloc verifică condiția dacă obținem valoarea true, adică în cazul nostru vom obține valoarea true ca atâta timp cât variabila k este mai mică de 5, atunci se execută comanda și anume codul cuprins între paranteze. După aceea, se execută al treilea bloc, adică o expresie matematică, în cazul nostru k++. Dacă obținem valoarea false în al doilea bloc, atunci bucla se termină. Iată un alt exemplu:

int a;
for(i=1; i<5;i++){
System.out.println(i);
}

Nu există practic nicio diferență cu bucla anterioară, dar în acest caz, putem folosi variabila în întreaga clasă, iar în prima opțiune, putem folosi variabila k doar în corpul buclei. Să trecem la al treilea exemplu:

int n, s;
for(n=1, s = 0; i>=s;s++, n++){
System.out.println(n, s);
}

Aici vedem declarația variabilelor n, s. Valorile au fost atribuite acestor variabile în buclă. În al doilea bloc, verificăm datele, adică am creat o condiție, iar în al treilea bloc executăm comenzi, dacă la verificarea în al doilea bloc obținem true (true). Este timpul de exemplu 4:

int s = 0, k = 1, n = 10;
for(; k<=n ;){
s+=k*k;
k++;
System.out.println(s, k, n);
}

Aici nu am declarat o variabilă, deoarece a fost deja creată și atribuită unui număr. În al doilea bloc am creat verificarea, iar în al treilea bloc nu am scris expresia matematică pentru că am scris-o în blocul de execuție a codului. Dacă al doilea bloc returnează adevărat, acesta este true. Să trecem la ultimul exemplu, când bucla este complet goală:

int s = 0, k = 1, n = 10;
for(; ;){
s++;
k++;
n++;
System.out.println(s, k, n);
}

Aceasta este o buclă infinită, ceea ce înseamnă că condiția va exista întotdeauna true.

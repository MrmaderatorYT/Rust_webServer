Bucla while

Iată codul standard de execuție a buclei while:

while (condiție){
echipe
}

While din engleză înseamnă până. Această buclă verifică starea din paranteze. Și dacă verificarea condiției devine true, comenzile sunt executate și apoi această buclă va începe din nou și comenzile vor fi executate din nou și așa mai departe. Pentru a ieși din buclă, trebuie să ieși din buclă, trebuie ca condiția să fie false

Iată un cod simplu care folosește o buclă:

int a = 0;
while (a<5){
System.out.println(a);
++a;
}

Vom primi cinci valori în consolă: 0,1,2,3,4. Am primit numărul 0, pentru că este începutul numărătorii inverse și se bifează și zero. Dacă codul este înlocuit cu locuri, adică.

int a = 0;
while (a<5){
++a;
System.out.println(a);
}

Vom obține cinci valori, dar răspunsul va fi diferit: 1,2,3,4,5. Mai întâi am adăugat valoarea și apoi o scoatem. Adică, primul număr este 0 și l-am adăugat și s-a dovedit a fi 1 și scoatem această unitate. La ultima verificare, am avut numărul 4, apoi adunat, apoi scăzut, iar în acest caz am obținut valoarea 5, deși condiția este strictă.

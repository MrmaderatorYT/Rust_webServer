HashMap

Luați în considerare o problemă în care trebuie să procesăm o mulțime de date... Milioane de informații. Și pentru asta este HashMap
HashMap - este un tablou asociativ. Dar este posibil să avem dificultăți în a găsi o persoană prin pașaport sau a introduce două persoane cu aceleași date de pașaport, ceea ce este imposibil. Și din nou, HashMap ne va ajuta aici.
Structura creării acestui tip de matrice este foarte asemănătoare cu scrierea unui ArrayList, unde cheia (Integer) vine pe primul loc, urmată de valoarea (String):

HashMap <Integer, String> hashmap = new HashMap<>();

Să adăugăm valori la HashMap datorită metodei put:

import java.util.HashMap;
public class Main{
public static void main(String[] args){
HashMap <Integer, String> hashmap = new HashMap<>();
hashmap.put(1, "Nume");
hashmap.put(2, "Nume1");
System.out.println(hashmap);
----------------------------
{1=Nume, 2=Nume1}

Iată și alte metode:
  • put() - adaugă un element la sfârșitul structurii
  • get() - returnează valoarea elementului specificat
  • size() - returnează numărul de elemente de structură
  • remove() - elimină elementul specificat din structură
  • containsKey() - verifică prezența cheii specificate
  • containsValue() - verifică prezența elementului specificat
  • clear() - elimină toate elementele din structură
  • isEmpty() - returnează prezența a cel puțin un element
  • putAll() - combină două hărți într-una singură

Nuanțe

Dacă dorim să creăm un nou element al structurii, dar nu specificăm o nouă cheie, ci doar o nouă valoare cu cheia creată - elementul va fi pur și simplu suprascris sub cheia corespunzătoare:

import java.util.HashMap;
public class Main{
public static void main(String[] args){
HashMap <Integer, String> hashmap = new HashMap<>();
hashmap.put(1, "Nume");
hashmap.put(2, "Nume1");
System.out.println(hashmap);
hashmap.put(1, "Nume1");
System.out.println(hashmap);
----------------------------
{1=Ім'я, 2=Nume1}
{1=Nume1, 2=Nume1}

Modificatorul strictfp

strictfp - modificator care limitează precizia constrângerilor cu float și duble prin format IEEE (Institutul de Ingineri Electrici). Acest lucru este necesar pentru a asigura portabilitatea. Faptul este că JVM-ul folosește toată precizia posibilă a procesorului și este diferit pe sisteme diferite, astfel încât rezultatul poate fi diferit. Acest modificator poate fi aplicat claselor, interfețelor și metodelor. Folosind acest modificator, vom obține același comportament atunci când manipulăm numerele în virgulă mobilă pe toate platformele.
clasa strictfp Main{
double num1 = 10e+102;
double num2 = 6e+08;
double calc(){
return num1+num2;
}

Într-o clasă, indică faptul că toate metodele de pe toate sistemele vor returnna numere cu aceeași precizie. Dacă numerele 0,112 și 0,113 sunt comparate cu cea mai apropiată sutime, acestea vor fi aceleași.

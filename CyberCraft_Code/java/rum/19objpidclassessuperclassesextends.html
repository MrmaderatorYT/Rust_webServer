Obiecte subclase și modificarea superclasei. Moștenire.

Voi începe cu moștenirea. Moștenirea este, să spunem, cel mai puternic mecanism de programare. Logica moștenirii este foarte simplă: atunci când se creează sau se descrie o clasă, o clasă exterioară, creată anterior sau, din nou, descrisă este luată ca bază. Acest lucru economisește timp, volumul codului și compatibilitatea codului. Cuvântul cheie extends:

este folosit pentru imitare

class subclasă extends Super clasa{
cod și așa mai departe
}

Să creăm o superclasă SuperClass și să o lăsăm goală. Și avem o subclasă Main și acum putem moșteni superclasa SuperClass:

class Main extends SuperClass{
cod și așa mai departe
}
class SuperClass{
}

Hai să mergem la superclasă. Dar să scriem o variabilă care va fi egală cu 1 cu o metodă care va scoate valoarea:

class Main extends SuperClass{
cod și așa mai departe
}
class SuperClass{
int number = 1;
void show(){
System.out.println("Numeric: "+number);
}
}

Să revenim la subclasa Main și să creăm o variabilă s de tipul char p>

class Main extends SuperClass{
char s;
}
class SuperClass{
int number = 1;
void show(){
System.out.println("Numeric: "+number);
}
}

Ми в підкласі Main спадкували суперклас SuperClass și datorită acesteia, subclasa Main descrie metoda show și variabila număr.
Conceptul de superclasă este relativ: o clasă poate fi o superclasă a unei clase și, în același timp, poate fi ea însăși o subclasă a unei alte clase. Acest tip de moștenire se numește moștenire pe mai multe niveluri.
Un subiect anterior a folosit cuvântul cheie this. Astăzi ni se alătură „antonimul” acestui cuvânt cheie, și anume - super
this - reprezintă instanța curentă a clasei.
super - reprezintă o superclasă.
Să creăm un exemplu:

class A{
А(){
System.out.println("Constructor fără argumente din clasa A");
}
A(String args){
System.out.println("Un constructor cu un argument din clasa A");
}
}
class B extends A{
B(){
this(""); // apelarea constructorului cu un argument din clasa B
System.out.println("Конструктор без аргументів класу В");
}
B(String args){
super(""); //apel cu un argument de clasa A
System.out.println("Un constructor cu un argument din clasa B"); }
}
public class Main{
public static void main(String args[]){
B b = new B();
}
}

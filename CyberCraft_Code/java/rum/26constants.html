Constante

Constantele pot fi folosite pentru a salva date. Variabilelor constante li se poate atribui o valoare o singură dată. Există un cuvânt cheie final pentru aceasta. De obicei, numele variabilelor constante sunt scrise cu majuscule, adică astfel:

final int NUMBER = 50

Dar puteți scrie din registrul inferior:

final int number = 50;

De obicei, constantele sunt statice, adică:

static final int a;

Dar ele sunt utilizate în general atunci când se declară în interfețe, deoarece trebuie întotdeauna inițializate direct în interfață.

Clasuri constante

Dacă clasa este constantă, atunci nu poate avea subclase, adică moștenirea este interzisă:

public final class String{
}
class String1 extends String{
}

În acest caz, va apărea o eroare. Există și o nuanță cu clasele abstracte, deoarece acestea sunt concepte care se exclud reciproc.
Folosind tablouri statice, nu putem modifica referința la obiect, dar putem modifica datele:

final int [ ] array = {1, 2, 3, 4, 5};
array[0] = 0;
array = new int [5]; // eroare

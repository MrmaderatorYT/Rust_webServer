Operator de selecție (switch-case)

Un operator de selecție cunoscut sub numele de switch-case permite executarea diferitelor blocuri de comenzi din valoarea unei expresii. O declarație de selecție începe cu cuvântul cheie switch. După aceea, se pun paranteze în care se pune o expresie, care va avea semnificații diferite. Să creăm un exemplu simplu:

int x=2;
switch(x){

Am scris aici o expresie pentru a verifica valorile din variabila x. După cum puteți vedea, am deschis aici parantezele pentru a verifica valorile. Avem nevoie de cuvântul cheie case pentru a valida valoarea. După cuvântul cheie, indicăm valoarea pe care o poate dobândi. Comparăm cu valoarea expresiei din variabila x. Deci, să scriem:

int x=2;
switch(x){
case 1:

Valoarea care trebuie comparată este urmată de două puncte. Acestea sunt așa-numitele paranteze, în care va fi executat codul. Să transmitem o valoare în consolă:

int x=2;
switch(x){
case 1:
System.out.println("Numărul 1");

După completarea descrierii codului containerului pentru o anumită valoare, de obicei este plasat cuvântul cheie break
break oprește execuția comenzii dacă obținem o potrivire de verificare. Să creăm pentru alte valori:

int x=2;
switch(x){
case 1:
System.out.println("Numărul 1");
break;
case 2:
System.out.println("Numărul 2");
break;
case 3:
System.out.println("Numărul 3");
break; -----------------------------
Numărul 2

Dar dacă trebuie să scoatem aceeași valoare pentru date diferite, de exemplu, ieșire către consolă: „Numărul 2” pentru valorile 2 și 1. Putem scrie în cod normal așa cum am făcut, înlocuind doar executarea codului, dar pentru a economisi spațiu și timp se poate face după cum urmează:

int x=3;
switch(x){
case 1:
System.out.println("Numărul 1");
break;
case 2:
case 3:
case 4:
System.out.println("Numărul 2");
break;
-----------------------------
Numărul 2

Dar dacă variabila capătă o valoare care nu există în blocul switch-case?
Cuvântul cheie default va ajuta noi pentru aceasta >

default (din engleză implicit) - va executa codul dacă valoarea variabilei nu se află în containere. Acest cuvânt cheie este plasat la sfârșit:

int x=100;
switch(x){
case 1:
System.out.println("Numărul 1");
break;
case 2:
System.out.println("Numărul 2");
break;
case 3:
System.out.println("Numărul 3");
break;
default:
System.out.println("Numărul N");
break;
-----------------------------
Numărul N

Există încă câteva variante de scriere a acestor blocuri, dar pentru versiuni ulterioare, dar aceasta este convenția standard.

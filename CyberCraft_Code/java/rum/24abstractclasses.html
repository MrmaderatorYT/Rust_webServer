Clase abstracte, metode

Metode abstracte

Metode abstracte - metode în care este posibil să te limitezi la declararea metodei, și anume să indice semnătura acesteia, dar nu să descrii corpul metodei. Fiecare metodă abstractă este descrisă de cuvântul cheie abstract. Metodele abstracte nu pot fi descrise deloc, ci doar declarate:

public abstract void one();

Абстрактні класи

Clasă abstractă - o clasă care conține cel puțin o metodă abstractă. Chiar dacă clasa este declarată cu cuvântul cheie abstract, va fi și abstractă. Pe baza claselor abstracte, este imposibil să se creeze un obiect, ceea ce ridică întrebarea logică: de ce sunt deloc? Clasele abstracte sunt de obicei folosite pentru moștenire. Cu cuvinte simple, pe baza unei clase abstracte, se creează subclase prin moștenire, în care metodele abstracte din superclasă sunt descrise cu cod de program. De asemenea, dacă unele metode abstracte sunt lăsate nedescrise într-o subclasă care este creată pe baza unei superclase abstracte, atunci o astfel de clasă este considerată abstractă:

public abstract class Main{
abstract void a();
}

Să revenim și la crearea obiectelor. Deși nu puteți crea o variabilă obiect bazată pe o clasă abstractă, puteți crea o variabilă obiect a unei clase abstracte. O astfel de variabilă se poate referi la un obiect al unei subclase create pe baza acestei clase abstracte. Să creăm o clasă abstractă:

public abstract class Main{
private int age;
public int getAge(){
return age;
}
}

Aici am creat o clasă abstractă, am înregistrat variabila vârstă. De asemenea, am creat funcționalitatea de bază a clasei. Datorită acestui lucru, putem urma această clasă abstractă pentru a descrie funcționalitatea. După cum am menționat mai sus, nu putem folosi constructorul unei clase abstracte pentru a crea un obiect bazat pe acesta, adică:

Main main = new Main(); // gresit

Iată un exemplu de clase și metode abstracte:

//clasa de forme abstracte
abstract class Figure{
float x;
float y;
Figure(float x, float y){
this.x = x;
this.y = y;
}
//metoda abstracta pentru a obtine perimetrul
public abstract float getPerimeter();
//metoda abstracta pentru a obtine zona
public abstract float getArea();
} //clasă derivată de dreptunghi
class Rectangle extends Figure{
private float width;
private float height;
//constructor care apelează constructorul clasei Figure
Rectangle(float x, float y, float width, float height){
super (x,y);
this.width = width;
this.height = height;
}
public float getPerimeter(){
return width * 2 + height * 2;
}
public float getArea(){
return width*height; }
}

Obiecte și clase

Să creăm o nouă clasă și o variabilă la sfârșitul clasei noastre principale:

public class Main{
public static void main(String[] args){
}
}
class One{
int a;
}

Care clasă este cea principală?

Clasa principală este cea care are scrisă metoda principală main. Adică, dacă noua clasă One ar avea această metodă, atunci metoda principală ar fi One. Acum să creăm un obiect în clasa principală:
Obiect sunt date grupate împreună și metode de procesare a acestor date. Când vorbim despre date, ne referim la variabile.

public class Main{
public static void main(String[] args){
One obj;
}
}
class One{
int a;
}

Unul este un așa-numit tip de date, dar un tip de date de clasă.
obj este un nume de variabilă.

Dar încă nu este un obiect. Am creat o variabilă bazată pe clasa One.
Pentru a crea un obiect, trebuie să utilizați cuvântul cheie nou pe următoarea linie pentru a spune că această variabilă aparține clasei One:

public class Main{
public static void main(String[] args){
One obj;
obj = new One();
}
}
class One{
int a;
}

Aceste două linii pot fi combinate într-una singură:

public class Main{
public static void main(String[] args){
One obj = new One();
}
}
class One{
int a;
}

Acum putem folosi variabile din clasa Una:

public class Main{
public static void main(String[] args){
One obj = new One();
obj.a=10;
String text = "Numeric: " + obj.a;
System.out.println(text); }
}
class One{
int a;
}

Asta e tot deocamdată

Operator instanceof

instanceof - operatorul care este necesar pentru a verifica dacă obiectul la care face referire variabila x a fost creat pe baza oricărei clase Y. Pe baza acestui lucru, putem înțelege că acest operator este boolean și returnează true și false. Să creăm un exemplu simplu:

public class Main{
public static void main(String[] args){
Integer x = new Integer(1);
System.out.println(x instanceof Integer);


------------
true

Am creat un obiect bazat pe class Integer și am scris o valoare acolo și apoi am creat o verificare: variabila x aparține clasei Integer? Și așa cum ar trebui să fie, am primit valoarea true. Dacă am creat o verificare pentru o altă clasă, am primit o eroare chiar înainte de compilare:

public class Main{
public static void main(String[] args){
Integer x = new Integer(1);
System.out.println(x instanceof String);


------------

Operatori logici. Operatori de comparație

Să începem cu operatorii de comparație. Sunt 6 în total

Să creăm cod standard pentru această lecție:

int a = 5;
int b = 6;
System.out.println(a b);
------------------------

Voi lăsa un spațiu între variabilele a și b și între ele voi pune operatori logici

< este operatorul mai puțin decât. Dacă variabila a este mai mică decât variabila b, true va fi transmisă în consolă. Dacă variabila a este mai mare decât b, atunci când utilizați acest operator - false< / span>. Iată un exemplu:

int a = 5;
int b = 6;
System.out.println(a>b);
------------------------
true

<= - operator mai mic sau egal cu. Dacă variabila a este mai mică sau egală cu b, atunci vom obține valoarea adevărat. De exemplu, să înlocuim valoarea a cu 6. Am primit valoarea true deci si in acesta. În alte cazuri - false.

int a = 5;
int b = 6;
System.out.println(a<=b);
-------------------------
true

> - cu atât mai mult operator. Dacă variabila a este mai mare decât b, atunci vom obține valoarea true span>, dar dacă a nostru este mai mic decât b, vom obține valoarea false

int a = 5;
int b = 6;
System.out.println(a>b);
------------------------
false

>= - operator mai mare sau egal. Valoarea true va fi trimisă în consolă dacă cel puțin o condiție este îndeplinită: dacă variabila a este mai mare decât b, sau că variabilele au aceleași valori.

int a = 5;
int b = 6;
System.out.println(a>=b);
-------------------------
false

== - operatorul este egal. Vom obține valoarea true numai dacă valorile a și b sunt la fel

int a = 5;
int b = 6;
System.out.println(a==b);
-------------------------
false

!= - operatorul nu este egal. Spre deosebire de declarația anterioară, vom obține valoarea true în toate cazurile în care a și b nu la fel

int a = 5;
int b = 6;
System.out.println(a!=b);
-------------------------
true

  • Dacă comparăm diferite tipuri numerice de variabile, iată un exemplu:

    int a = 5;
    double b = 5;
    System.out.println(a==b);
    -------------------------
    true

    De asemenea, dacă adăugați zerouri după cele cinci din variabila b, compilatorul va elimina toate zerourile, cu excepția primei, adică:

    double a = 5.00000;
    System.out.println(a);
    ---------------------
    5.0

    Și, deoarece 5.0 este 5, verificarea cu int va returna true, ceea ce era înainte.

  • Compararea tipurilor numerice cu tipurile de text este imposibilă!

Să trecem la tipurile logice.

& - operator binar „logic și”. Această declarație va returna true dacă valorile variabilelor a și b span> sunt egale cu true, dacă cel puțin o variabilă are valoarea false, atunci vom obține și valoarea false

boolean a = true;
boolean b = false;
System.out.println(a&b);
---------------------
false

&& este un operator binar „logic și” care utilizează o schemă simplificată. Nu există nicio diferență cu răspunsul afirmației anterioare, dar această instrucțiune verifică prima variabilă și dacă este egală cu false, atunci următoarele variabile nu vor fi verificate, ci vor să fie afișat imediat în consolă false

| - logic binar sau operator. Vom obține valoarea true dacă valoarea a cel puțin uneia dintre variabile este true. Dacă valoarea tuturor operanzilor este egală cu false, atunci în consolă vom primi valoarea false

boolean a = true;
boolean b = false;
System.out.println(a | b);
---------------------
true

| | este un operator binar „logic sau”. Care folosește o schemă simplificată, rezultatul expresiei cu operatorul anterior nu diferă. Dar dacă cel puțin o valoare este egală cu true, atunci valoarea true este trimisă imediat în consolă. Dacă valoarea tuturor operanzilor este egală cu false, atunci în consolă vom primi valoarea false

^ - operator unar „includere logică sau”. Rezultatul va fi adevărat dacă o variabilă este true, iar cealaltă este false>

boolean a = true;
boolean b = false;
System.out.println(a^b);
---------------------
true

! - operator unar de „negație logică”. Rezultatul expresiei de evaluare !a este valoarea true dacă valoarea variabilei a este false

boolean a = true;
System.out.println(!a);
---------------------
false

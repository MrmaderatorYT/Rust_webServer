Enumerare

Enum este un tip de date special care permite unei variabile să fie un set de constante predefinite. Variabila trebuie să fie egală cu una dintre valorile sale predefinite. Un exemplu tipic este direcția orizontului: sensul nord, sud, vest, est. Sunt constante, deci pot fi scrise, deși nu neapărat, cu majuscule. Iată un exemplu:

enum public Compass{
NORTH, SOUTH, WEST, EAST;
}

De asemenea, enum trebuie să aibă un modificator de acces. Dacă enum nu este imbricat într-o clasă, atunci trebuie să fie public. Dacă enum va fi creat ca fișier separat, modificatorul de acces private nu poate fi utilizat. Dar, dacă enum este imbricată într-o clasă, aceasta poate fi privată:

public class Main{

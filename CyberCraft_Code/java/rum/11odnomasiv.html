Matrice unidimensională

Masiv - este o secvență de elemente de același tip unite printr-un nume comun.
Declararea unei matrice este ca o variabilă normală, dar după identificatorul de tip, adică int, < span id="name">char , byte și altele. Acest lucru permite determinarea tipului de elemente ale matricei, apoi sunt specificate paranteze pătrate goale, adică forma standard a unei matrice unidimensionale este următoarea:

tastați[ ] numele variabilei;

Este posibil să nu fie specificate paranteze pătrate goale după identificatorul de tip, adică:

introduceți numele variabilei[ ];

Aici nu este deloc diferență. Prima opțiune, adică, de exemplu:

int[ ] a;

Nu diferă de a doua opțiune, adică, de exemplu:

int a [ ];

În majoritatea cazurilor, vom folosi prima opțiune, deoarece a fost creată special pentru Java. Adică să aibă un stil de scriere, ca în Java. Iar al doilea este pentru programatorii care au trecut de la limbajul de programare C la Java. Și pentru a face mai convenabil pentru hackeri să scrie cod, au făcut această opțiune.
Pentru a crea o matrice, utilizați operatorul new
new - alocă memorie pentru un obiect, în cazul nostru pentru o matrice.
Dimensiunea matricei - numărul de elemente din matrice.
Mărimea matricei este indicată între paranteze drepte după identificatorul de tip al elementelor matricei, deci avem:

new a;

Această linie creează o matrice. O referință la o matrice este creată după cum urmează:

tip[ ] variabilă;
variabilă = new tip [ mărime ];

În prima linie am creat o matrice și i-am dat un nume și un tip, pe a doua linie am alocat memorie pentru această matrice. Aceste două linii pot fi combinate într-una singură:

type[ ] variabilă a = new tip [size];

Să creăm un exemplu:

int[ ] arr;
arr = new int [10];

În această linie, am alocat memorie pentru 10 elemente de tip int. Aceste două linii pot fi din nou combinate într-una singură:

int[ ] arr = new int [10];

Acum să trecem la punctul cheie:

Primul element al matricei are valoarea 0.

Acum, dacă trebuie să avem o valoare când creăm o matrice. Poate fi creat după cum urmează:

type[ ] variabilă a = {date}

Iată unul dintre exemple:

String [ ] cats = {"cat1", "cat2", "cat3", "cat4"};

Am creat o matrice de pisici și am specificat valorile între acolade, între ghilimele și virgule. Am creat o matrice de tip String și astfel putem crea o matrice de tip int

int [ ] cats = {10, 20, 30};

Pentru a obține un element de matrice, trebuie să vă referiți la numărul de index. Mai întâi, să creăm o matrice și apoi să scoatem textul:

String [ ] cars = {"BMV", "Ford", "Mazda"};
System.out.println(cars[0]);
-------------------------
BMV

Acum, să trecem la modificarea elementelor de date:

int [ ] nums = {10, 20, 30, 40};
cars[0] = 20;
System.out.println(nums[0]);
-------------------------
BMV

Am înlocuit primul element al matricei, adică numărul 10 cu numărul 20. Dar a fost scris că există un cuvânt cheie new, adică astfel:

String [ ] cars = new {"BMV", "Ford", "Mazda"};
System.out.println(cars[0]);
-------------------------
BMV

Nu va exista nicio diferență între ele, dar în cazul fără new nu putem scrie așa:

int[ ] arr;
arr = {10, 20, 30};

Tipul de matrice trebuie să fie întotdeauna indicat, adică ca în a doua opțiune:

arr = new int{10, 20, 30};

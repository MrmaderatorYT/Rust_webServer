Модифікатор strictfp

strictfp - модифікатор, що обмежує точність обмежень з float та double за форматом IEEE (інститут інженерів з електротехніки). Це потрібно, щоб забезпечити перенесення. Справа в тому, що JVM використовує всю можливу точність процесора, а вона на різних системах різна, тому і результат може вийти різний. Цей модифікатор може застосовуватись до класів, інтерфейсів та методів. При використанні цього модифікатора ми отримаємо однакову поведінку в ході маніпуляцій з числами з плаваючою крапкою на всіх платформах.
strictfp class Main{
double num1 = 10e+102;
double num2 = 6e+08;
double calc(){
return num1+num2;
}

У класі він позначає, що всі методи на всії системах повертатимуть числа з однаковою точністю. Якщо числа 0.112 та 0.113 порівнювати з точністю до сотих, то вони будуть однаковими.

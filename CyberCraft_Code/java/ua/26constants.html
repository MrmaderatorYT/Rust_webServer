Константи

Для збереження даних, можна використовувати константи. Константним змінним можна присвоїти значення один раз. Для цього є ключове слово final. Зазвичай назви константної змінної пишуться з великого реєстра, тобто так:

final int NUMBER = 50

Але можна й записувати з нижнього реєстра:

final int number = 50;

Зазвичай константи - статичні, тобто:

static final int a;

Але їх в загалом використовують при оголошенні в інтерфейсах, бо їх завжди потрібно проініціалізувати безпосередньо в інтерфейсі.

Константні класи

Якщо клас - константний, то він не зможе мати підкласи, тобто заборонено спадкування:

public final class String{
}
class String1 extends String{
}

В такому випадку буде помилка. Також є нюанс з абстрактними класами, боо це взаємновиключні поняття.
Використовуючи статичні масиви ми не зможемо змінювати посилання на об'єкт, але можемо змінювати дані:

final int [ ] array = {1, 2, 3, 4, 5};
array[0] = 0;
array = new int [5]; // помилка

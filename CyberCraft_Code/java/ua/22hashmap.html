HashMap

Розглянемо задачу, коли нам потрібно обробляти безліч даних... Мільйони даних інформації. І для цього є HashMap
HashMap - це асоціативний масив. Але в нас може виникнути складність пошуку людини по паспорту чи внесення двох людей з однаковими паспортними даними, що є неможливим. І знову ж, тут нам допоможе HashMap.
Структура створення цього типу масиву дуже схожа з написанням ArrayList, де на першому місці ключ (Integer), далі значення (String):

HashMap <Integer, String> hashmap = new HashMap<>();

Додамо значення в HashMap завдяки методу put:

import java.util.HashMap;
public class Main{
public static void main(String[] args){
HashMap <Integer, String> hashmap = new HashMap<>();
hashmap.put(1, "Ім'я");
hashmap.put(2, "Ім'я1");
System.out.println(hashmap);
# {1=Ім'я, 2=Ім'я1}

Ось інші методи:
  • put() - додає елемент в кінці структури
  • get() - повертає значення вказаного елементу
  • size() - повертає кількість елементів структури
  • remove() - видаляє вказаний елемент зі структури
  • containsKey() - перевіряє наявність вказаного ключа
  • containsValue() - перевіряє наявність вказаного елемента
  • clear() - видаляє всі елементи зі структури
  • isEmpty() - повертає наявність хоча б одного елементу
  • putAll() - поєднує два мапа в один

Нюанси

Якщо ми захочемо створити новий елемент структури, але не вкажемо новий ключ, а тільки нове значення зі створеним ключем - елемент просто перезапише під відповідним ключем:

import java.util.HashMap;
public class Main{
public static void main(String[] args){
HashMap <Integer, String> hashmap = new HashMap<>();
hashmap.put(1, "Ім'я");
hashmap.put(2, "Ім'я1");
System.out.println(hashmap);
hashmap.put(1, "Ім'я1");
System.out.println(hashmap);
{1=Ім'я, 2=Ім'я1}
{1=Ім'я1, 2=Ім'я1}

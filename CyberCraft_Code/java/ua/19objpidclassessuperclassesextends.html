Об'єкти підкласу та зміна суперкласу. Спадкування.

Розпочну зі спадкування. Спадкування - скажімо так, найпотужніший механізм програмування. Логіка спадкування дуже проста: під час створення або опису класу за основу береться внший, раніше створений або знову ж таки, описаний клас. Це дозволяє економити час, об'єм програмного коду й сумісність програмних кодів. Для наслідування використовується ключове слово extends:

class підклас extends Суперклас{
код і так далі
}

Давайте створимо суперклас SuperClass і просто залишемо його пустим. І у нас є підклас Main і тепер можна успадкувати суперклас SuperClass:

class Main extends SuperClass{
код і так далі
}
class SuperClass{
}

Перейдімо до суперкласу. Та запишемо якусь змінну, яка буде дорівнювати 1 з методом, який буде виводити значення:

class Main extends SuperClass{
код і так далі
}
class SuperClass{
int number = 1;
void show(){
System.out.println("Число: "+number);
}
}

Повернімося до підкласу Main і створимо змінну s типу char

class Main extends SuperClass{
char s;
}
class SuperClass{
int number = 1;
void show(){
System.out.println("Число: "+number);
}
}

Ми в підкласі Main спадкували суперклас SuperClass і завдяки цьому, в підкласі Main описано метод show та змінна number.
Поняття суперкласу є відносним: деякий клас може бути суперкласом для одного класу, і в той же час сам бути підкласом іншого класу. Цей тип спадкування називається багаторівневим спадкуванням.
На минулій темі було використано ключове слово this. Сьогодні до нас приєднується "антонім" до цього ключового слова, а саме - super
this - представляє поточний екземпляр класу.
super - представляє суперклас.
Створимо приклад:

class A{
А(){
System.out.println("Конструктор без аргуметів класу А");
}
A(String args){
System.out.println("Конструктор з одним аргументом класу А");
}
}
class B extends A{
B(){
this(""); // виклик конструктора з одним аргументом класу B
System.out.println("Конструктор без аргументів класу В");
}
B(String args){
super(""); //виклик аргументом класу А
System.out.println("Конструктор з одним аргументом класу В"); }
}
public class Main{
public static void main(String args[]){
B b = new B();
}
}

Об'єкти та класи

Створімо новий клас та змінну в кінці нашого класу Main:

public class Main{
public static void main(String[] args){
}
}
class One{
int a;
}

Який клас є головним?

Головний клас - це той, у кого записано головний метод main. Тобто, якщо в новому класі One мав цей метод, тоді головний метод був би One. Тепер в головному класі створимо об'єкт:
Об'єкт - це згруповані разом дані та методи для того, щоб ці дані обробляти. Коли говориться про дані, мається на увазі змінні.

public class Main{
public static void main(String[] args){
One obj;
}
}
class One{
int a;
}

One - так званний тип даних, але тип даних класу.
obj - назва змінної.

Але це не об'єкт, поки що. Ми створили змінну на основі класу One.
Щоб створити об'єкт, треба на наступному рядку, завдяки ключовому слову new сказати, що ця змінна належить класу One:

public class Main{
public static void main(String[] args){
One obj;
obj = new One();
}
}
class One{
int a;
}

Ці два рядки можна об'єднати в один:

public class Main{
public static void main(String[] args){
One obj = new One();
}
}
class One{
int a;
}

Тепер ми можемо використовувати змінні класу One:

public class Main{
public static void main(String[] args){
One obj = new One();
obj.a=10;
String text = "Число: " + obj.a;
System.out.println(text); }
}
class One{
int a;
}

Поки що це все

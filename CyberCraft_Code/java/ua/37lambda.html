Лямбда вирази

Лямбда вирази - певний блок коду, котрий визначає дію. Також можна розглядати лямбда-вирази, як визначення анонімного методу (наступний урок), тобто методу в якому немає імені. Також у них немає типу результату. Взагалі, лямбда-вирази - це гарна протидія анонімного класу, тож краще використовувати лямбда-вирази. Блок із команд у лямбда-виразів називається тіло лямбда-виразу. Відповідно лямбда-вираз складається з аргументів тіла. Цей вираз так само створюється, як звичайний, але тут немає назви та ідентифікатора типу результату. Команди в тілі лямбда-виразу описуються командами (ага, логіка №1). Для аргументів вказується тип (хоч і в деяких випадках не вказується). Команди пишуться у фігурних дужках і між аргументами і блоком команд вказується "стрілка": ->. І виходячи з цього, можемо отримати наступне:

(аргументи) -> {
команди
}
А тепер перейдімо стандартний вигляд, скажімо такий:
(int x, int y) -> {
int z = x+y;
System.out.println(z);
}

Існують деякі правила, які дозволяють спрощувати синтаксис описання лямбда-виразів:

  • Тип аргументу можна не вказувати, якщо він може бути ідентифікований перед лямбда-виразу
  • Якщо в лямбда-виразі один аргумент, тип якого не вказаний, то круглі дужки можна не використовувати
  • Якщо в лямбда-виразі немає аргументів, то використовується пусті круглі дужки.
  • Якщо в тілі лямбда-виразу всього одна команда, то фігурні дужки можна не використовувати. У цьому випадку в тілі лямбда-виразу складається з return-інструкція, але це ключове слово можна не використовувати

Функціональні інтерфейси

Функціональні інтерфейси - це інтерфейси з одним і тільки одним абстрактним методом. Тобто нічого особливого немає. Просто має бути виконана одна умова: в інтерфейсі оголошується один абстрактний метод. Особливість фунціонального інтерфейсу в плані використання лямбда-виразів пов'язано щ тим, що лямбда-вираз може бути присвоєним значенням змінній інтерфейсного типу - але за умови, що інтерфейс фунціональний, а сиґнатура абстрактного методу у фунціональному інтерфейсі відповідає параметрам лямбда-виразу - створюється об'єкт на основі анонімного класу, який реалізує даний інтерфейс, а як код для визначення абстрактного методу використовується код лямбда-виразу

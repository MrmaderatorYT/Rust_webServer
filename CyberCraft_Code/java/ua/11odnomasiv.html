Одновимірний масив

Масив - це послідовність однотипних елементів, об'єднаних загальним іменем.
Оголошення масиву подібно до звичайної змінної, але після ідентифікатора типу, тобто int, char, byte та інші. Це дає для визначення типу елементів масиву, потім вказуються пусті квадратні дужки, тобто стандартний вигляд одновимірного масиву такий:

тип[ ] ім'я змінної;

Пусті квадратні дужки можна вказувати не після ідентифікатора типу, тобто:

тип ім'я змінної[ ];

Тут різниці взагалі нема. Перший варіант, тобто, наприкад:

int[ ] a;

Не відрізняється від від другого варіанту, тобто наприклад:

int a [ ];

В більшості випадках ми будемо використовувати перший варіант, бо він був створений спеціально для Java. Тобто, щоб був стиль написання, як в Java. А другий - для програмістів, які перейшли з мови програмування C на Java. І для того, щоб сішерам було зручно писати код - зробили цей варіант.
Для створення масиву використовують оператор new
new - виділяє пам'ять під об'єкт, в нашому випадку під масив.
Розмір масиву - кількість елементів в масиві.
Розмір масиву вказується у квадратних дужках після ідентифікатора типу елементів масиву, отже маємо:

new a;

Завдяки цьому рядку створюється масив. Посилання на масив створюється так:

тип[ ] змінна;
змінна = new тип [розмір];

В першому рядку ми створили масив та дали йому назву та тип, на другому рядку ми виділили пам'ять на цей масив. Ці два рядки можна поєднати в один:

тип[ ] зміннa = new тип [розмір];

Створимо приклад:

int[ ] arr;
arr = new int [10];

В цьому рядку ми виділили пам'ять на 10 елементів типу int. Ці два рядки знову ж таки можна об'єднати в один:

int[ ] arr = new int [10];

Тепер перейдімо до ключового момента:

Перший елемент масиву має значення 0.

Тепер, якщо нам необхідно мати значення при створенні масиву. Це млжна створити наступним чином:

тип[ ] зміннa = {дані}

Ось один з прикладів:

String [ ] cats = {"cat1", "cat2", "cat3", "cat4"};

Ми створили масив cats та вказали значення у фігурних дужках у лапках та через кому. Це ми створили масив типу String і таким чином можна створити масив типу int

int [ ] cats = {10, 20, 30};

Щоб отримати елемент масиву, треба посилатися на номер індексу. Для початку створимо масив, а потім виведемо текст:

String [ ] cars = {"BMV", "Ford", "Mazda"};
System.out.println(cars[0]);
//BMV

Тепер перейдімо до зміни даних елементів:

int [ ] nums = {10, 20, 30, 40};
cars[0] = 20;
System.out.println(nums[0]);
BMV

Ми замінили перший елемент масиву, тобто число 10 на число 20. Але було написано що є ключове слово new, тобто так:

String [ ] cars = new {"BMV", "Ford", "Mazda"};
System.out.println(cars[0]);
20

Різниці між ними не буде, але в випадку без new ми не можемо записати так:

int[ ] arr;
arr = {10, 20, 30};

Завжди повинен вказуватися тип масиву, тобто як в другому варіанті:

arr = new int{10, 20, 30};

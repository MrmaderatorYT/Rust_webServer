Цикл while

Ось вам стандартний код виконання циклу while:

while (умова){
команди
}

While з англійської означає поки. Цей цикл перевіряє умову, яка в дужках. І якщо перевірка умови отримає значення true, виконуються команди, а потім цей цикл знову запуститься і знову буде виконуватися команди і так далі. Щоб вийти з циклу, потрібно, щоб вийти з циклу, потрібно, щоб умова була false

Ось простий код з використанням циклу:

int a = 0;
while (a<5){
System.out.println(a);
++a;
}

Ми отримаємо в консолі п'ять значень: 0,1,2,3,4. Ми отримали число 0, бо це початок відліку і нуль також перевіряється Якщо код замінити місцями, тобто.

int a = 0;
while (a<5){
++a;
System.out.println(a);
}

Ми отримаємо п'ять значень, але відповідь буде інша: 1,2,3,4,5. Ми спочатку додали значення, а потім ми його виводимо. Тобто перше число - 0 і ми його додали і вийшло 1 і цю одиничку ми виводимо. В останній перевірці ми мали число 4, потім додали, а потім виводили і в цьому випадку ми отримали значення 5, хоч умова строга.

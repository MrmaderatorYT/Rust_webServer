Логічні оператори. Оператори порівняння

Почнемо з операторів порівняння. Всього їх 6

Створімо стандартний код для цього уроку:

int a = 5;
int b = 6;
System.out.println(a b);

Я залишу пустоту між змінними a та b та між ними я буду ставити логічні оператори

< - це оператор менше. Якщо змінна а буде менша за змінну b, в консоль виведеться true. Якщо змінна a буде більша за b, то при використанні цього оператора - буде виведено false. Ось приклад:

int a = 5;
int b = 6;
System.out.println(a>b);
true

<= - оператор менше або дорівнює. Якщо змінна a менше або дорівнює b, то отримаємо значення true. Давайте для прикладу замінемо значення a на 6Ми отримали і впершому випадку значення true так і в цьому. В інших випадках - false.

int a = 5;
int b = 6;
System.out.println(a<=b);
true

> - оператор більше. Якщо змінна a буде більша за b, то отримаємо значення true, але якщо в нас a менше за b, отримаємо значення false

int a = 5;
int b = 6;
System.out.println(a>b);
false

>= - оператор більше або дорівнює. Виводиметься в консоль значення true, якщо хоча б одна умова виконуєтьється: чи те, що змінна a більше за b, чи те, що змінні мають однакові значення.

int a = 5;
int b = 6;
System.out.println(a>=b);
false

== - оператор дорівнює. Ми отримаємо значення true, тільки в тому випадку, якщо значення a та b однакові

int a = 5;
int b = 6;
System.out.println(a==b);
false

!= - оператор не дорівнює. На відміну від попереднього оператора, ми отримаємо значення true в усіх випадках, коли a та bне однакові

int a = 5;
int b = 6;
System.out.println(a!=b);
true

  • Якщо ми будемо порівювати різні числові типи змінних, ось для прикладу:

    int a = 5;
    double b = 5;
    System.out.println(a==b);
    true

    Також, якщо додати в змінній b нулі після п'ятірки, то компілятор прибере всі нулі, окрім першої, тобто:

    double a = 5.00000;
    System.out.println(a);
    5.0

    І так як 5.0 це 5, при перевірці з int ми отримаємо true, що було раніше.

  • Порівняння числових типів з тексторвими неможливе!

Перейдімо до логічних типів.

& - бінарний оператор "логічне і". Цей оператор виведе true, якщо значення змінних a та b дорівняють true, якщо хоча б одна змінна має значення false, то й в консолі отримаємо значення false

boolean a = true;
boolean b = false;
System.out.println(a&b);
false

&& - бінарний оператор "логічне і", який користується спрощеною схемою. Різниці з відповідю попереднього оператора нема, але цей оператор перевіряє першу змінну і якщо вона буде дорівнювати false, то перевірка наступних змінних не буде, а відразу буде виведено в консолі false

| - бінарний оператор "логічне або". Ми отримаємо значення true, якщо значення хоча б одного зі змінних дорівнює true. Якщо значення усіх операндів дорівнює false, то в консолі отримаємо значення false

boolean a = true;
boolean b = false;
System.out.println(a | b);
true

| | - бінарний оператор "логічне або". Який користується спрощеною схемою, Результатом виразу з попереднім оператором не відрізняється. Але якщо хоча б одне значення дорівнює true, то в консоль відразу виводиться значення true. Якщо значення усіх операндів дорівнює false, то в консолі отримаємо значення false

^ - унарний оператор "логічне включення або". Результатом буде true, якщо одна змінна має значення true, а інша false

boolean a = true;
boolean b = false;
System.out.println(a^b);
true

! - унарний оператор "логічного заперечення". Результатом виразу розгляду !а є значення true, якщо значення змінної a дорівнює false

boolean a = true;
System.out.println(!a);
false

Динамічний масив (ArrayList)

ArrayList - динамічний масив, може змінювати свою довжину. Коли цей внутрішній масив заповнюється, ArrayList створює новий масив, копіює дані зі старого, записує в новий та збільшує розмір. Ось формула розміру нового масиву:
Розмір нового масиву = (розмір старого масиву * 1.5)+1
Після запису даних в новий масив, старий видаляється. Ось стандартний приклад:

ArrayList list = new ArrayList();

Це ми створили динамічний масив з навою list. На відміну від інших масивів, ми не вказуємо тут тип даних, бо динамічні масиви можуть приймати будь-які типи даних, хоч дані з іншого класу. І тут ми перейдемо до дженериків:
Дженерики - це типи з параметром. B Java, класи-контейнери дозоляють вказувати їх внутрішні об'єкти. Коли ми оголошуємо дженерик-змінну, то вказуємо не один тип, а два:

  1. тип змінної
  2. тип даних

ArrayList<String>
ArrayList<Integer>

Як ви бачите, ми вказали тип даних у трикутних дужках, відомі як символи більше та менше. Чому ж ми не записали в дужках звичайний тип даних, як int та інші? Бо, ми звертаємося до класу-оболонки типу. Ось приклад:

ArrayList <String> list = new ArrayList<>();
list.add("Hi");

Ми створили динамічний масив типу String з назвою list. На наступному рядку, ми використали метод add, який додає значення в масив. Але додає це значення в кінець. Також є інші методи для ArrayList.
  • add(індекс, елемент) - цей варіант записує значення вказаного елемента. Значення індексу - порядковий номер для запису. Якщо не вказати індекс, буде доданий елемент в кінець.
  • get() - повертає значення вказаного елементу.
  • contains() - перевіряє чи містить вказаний елемент в масиві.
  • remove() - видаляє елемент з масиву.
  • clear() - очищує масив, робить його розмір нуль.
  • indexOf() - повертає індекс шуканого елемента.

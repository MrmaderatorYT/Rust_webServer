Статичні блоки

Статичні блоки використовуються для статичної ініціалізації класу. Цей код всередині статичного блоку виконується тільки один раз: коли клас завантажується в пам'ять вперше і це виглядає так:

static{
System.out.println("Статичний блок");
}

Також статичний блок буде працювати без методу main, якщо використовувати версію JDK 1.6 і більш пізнішої версії (1.5, 1.4 і т.д.):

class Main{
static{
System.out.println(Воно працює! :0");
}
}

В інакшому випадку, тобто для версії 1.7 та вище - ми отримаємо помилку. Також ці методи не маєть назви та модифікатору доступу, тож викликати їх не можна і будуть виконуватися тільки в цьому класі. Також при наслідуванні, статичні блоки не будуть виконуватися під час запуску програми.

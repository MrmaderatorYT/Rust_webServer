Методи

Метод - це блок коду, який виконується тільки тоді, коли викликається. Також інша назва методів - функції.

Навіщо методи?

Методи потрібні для повторного використання коду: напиши один код в методі та викликай його, коли потрібно, а не переписувати. Ось приклад:

public class Main{
public static void main(String[] args){
код
}
public static void method(){
код
}
}

Ось стандартний метод. method - назва методу.
public static void ми будемо вивчати в майбутньому, але для розуміння я поясню:

public - модифікатор, який вказує, що цей метод можна використовувати будь-де.

static - модифікатор, який застосовується до поля блоку, методу. Він вказує на прив'язку на прив'язку суб'єкта. В нашому випадку - метода.

void - по-перше, з англійської - це пустота, по-друге, void - це тип даних, який ми можемо отримати в результаті методі. Void - це тип даних, що і int та інші. Це ключове слово використовує для того, щоб повертати змішані дані, тобто в одночас String та double й інші. Створимо простий метод:

public static void method(){
System.out.println("Текст з іншого методу");
}

А тепер викличемо цей метод в головному методі main

public class Main{
public static void main(String[] args){
method();
}
public static void method(){
System.out.println("Текст з іншого методу");
}
}
Текст з іншого методу

Але так буває, що нам потрібно передавати якусь інформацію в метод. В цьому нам допоможуть пареметри, які ми запишемо в дужках:

public static void method(String name, int age){
System.out.println("My name: " + name);
System.out.println("My age: " + age);
}

І тепер викличемо цей метод в головному методі:

public class Main{
public static void main(String[] args){
method("Ann", 15);
}
public static void method(String name, int age){
System.out.println("My name: " + name);
System.out.println("My age: " + age);
}
}
// My name: Ann
// My age: 15

В головному методі викликали метод method та записали туди значення. Але є нюанс - виклик методу повинен мати туку ж кількість аргументів та послідовність типів як в описі, тобто ми створили метод в якому записали два типи даних - String, int. При виклику метода, ми повинні вказати їх, тобто спочатку ми вказуємо тип String, а потім int
Перейдімо до повернення певних значень. Замість ключового слова void ми повинні вказати потрібний нам тип, нехай буде String.

public class Main{
public static void main(String[] args){
String s = "Hi";
s = method(s);
System.out.println(s);
}
static String method(String str){
str=str+"world";
return str;
}
}
// Hi world

В цьому коді ми створили змінну s типу String та записали текст. В методі method ми замінили аргументи на один - str типу String. ми привоїли змінній str до пустого рядка текст world, а завдяки ключовому слову return ми повертаємо дані змінної str. В головному методі ми присвоємо змінній s метод method та в дужках ми запишемо змінну s, яку будемо записувати в метод. На наступному рядку ми виводимо в консоль змінну s.

return

return - ключове слово, яке допомагає повертати значення. Давайте не поспішати і створимо стандартний шаблон проєкту:

public class Main{
public static void main(String[] args){
System.out.println(Hello world");
}
}

Як нам відомо, шо метод main є точкою старту програми. Цей метод неможе повертати значення, тому створимо новий метод:

public class Main{
public static void main(String[] args){
System.out.println(massage());
}
public static String massage(){
return "Hello world";
}

Як ми бачимо, ми створили метод massage та змінили метод main - під час виведення ми викликаємо метод. І в консолі ми отримаємо Hello world. Завдяки ключовому слову return ми вказали значення, яке повинні повернути. Ми вказали, що метод має тип String і тому він поверне значення String. Це допомоєе компілятору перевірити, що дії методу погоджується таким образом, як він оголошеним. І якщо ми будемо повертати інше значення в цьому методі, наприклад типу int, або інший навідміну з тим типом який ми вказали в методі, отримаємо помилку, що це не сумісні типи.

public static String massage(){
return 5; //помилка
}

Також, цей тип відноситься до операторів керування коду. Думаю всі знають, що код виконується зверху вниз, але цей оператор має можливість порушити виконання коду:

public class Main{
public static void main(String[] args){
if(args.lenght == 0){
return;
}
for(String arg : args){
System.out.println(arg);
}
}
}

Як бачимо, що в методі main немає параметрів перед перевірку і тому елементів у нас нуль і після цього виконується перевірка, котра зупиняє виконання цього методу (в нашому випадку main)
args - змінна, яку ми створили в параметрах методу main як масив і тому ми можемо в перевірці використати метод lenght, який використовується для масивів.

Типи данних

Типи даних - можливість приймати змінній саме ті значення, які вказали. В Java існує два типи даних - примітивний та вказівникові.

Всього існує 8 типів примітивних данних

  1. byte - тип, який приймає тільки цілі числа. Діапазон чисел становить від -128 до 127 (число 0 також тут) витрачає вісім біт пам'яті.

    2. byte a = 120;

  2. short - також може приймати тільки цілі числа, але діапазрн значно більший: від -32768 до 32767, але так як діапазон більший, то і використання пам'яті буде більшим, а саме 16 біт

    3. short b = 20;

  3. int - приймає тільки цілі числа в діапазоні від -2 147 483 648 до 2 147 483 647 та займає 32 біта пам'яті

    4. int a = 203452;

  4. long - які попередні, приймає цілі числа та займає 64 біта ппам'яті. Приймає діапазон від -9223372036854775808 до 9223372036854775807
    ! Якщо ввести значення більші за діапазон int, тобто грубо кажучи більше 3-х мільйонів буде помилка. Помилка в тому, що Компілятор сприймає значення long за int. Щоб полагодити помилку, потрібно додати лише велику літеру L
    L - скорочення від long. Ми пишемо з великої літери, бо ми звертаємося до класу-оболонки. У кожно типу є свій клас-оболонка, наприклад, int має клас оболонку integer.

    5. long a = -15023125;

  5. float - приймає тільки числа в діапазоні від 3.4*10^-38 до 3.4*10^38 цей тип може приймати як і цілі, так і дробові числа. Займає 32 біта пам'яті
    Як і long, float'у потрібно додаввти клас-оболонки. У float - це... Той самий float, але з феликої літери, тобто F

    6. float a = -1.5;

  6. double - приймає тільки числа в діапазоні 1.7*10^308 по модулю та займає 64 біта пам'яті. Різниця між double та float полягає не тільки в діапазані, а в точності. Наприклад, float може отримати значення: 1.6893, а double1.6894, бо ми окрунлили до десятитисячеих, а так як doubleсприймає більше значень після коми за float, то дані будуть більш точніші.

    7. double a = -15023125.225484;

  7. char - символьний тип. Може сприймати букви. Розмір пам'яті виділяється 16 біт.
    char позначається в лапках, але лапки одинарні, тобто ось: ' ' в цих лапках можна поставитилише один символ, це як звичайний пробіл, цифру, бувку. Коротше кажучи, любий символ з таблиці Unicode, відома ще як utf-8.

    8. char a = 'q';

  8. String відразу кажу, що цей тип пишеться з великої літери, по друге, текст записується в подвійних лапках " ". Також в цей тип можна записати безліч символів, тобто на відміну від char ми можемо записати сюди хоч "Енеїду" Котляревського.

    9. String a = "Glory to Ukraine";

    Більшість компіляторів не позначають кольором String так як я казав, що це не тип данних. В коді я також не буду позначати String кольором

  9. boolean - логічний тип. Сприймає тільки два значення, а саме true та false

    10. boolean a = true;

Я казав, що всього існує 8 типів данних, а тут 9... Комедія в тому, що String не є примітивним типом данних. Це вказівниковий тип, тобто об'єкт. Примітиви виділяються іншим кольором, ніж звичайний чорний та білий. Перейдімо до приведення типів


Приведення типів


Тут пи будемо використовувати операнд
Операнд - аргумент операції. Приведення типів поділяють на два типи: явне та неявне(або автоматичне). Явне і неявне приведення типів проводиться автоматично по певній лозіці: якщо в числовому виразі є операнд типу double, то всі інші операнди, резулятити виразу автоматично розширується до типу double

double a2 = 0.02;
a=(int)a

Двовимірний масив

Створення двовимірного масиву дуже схоже з написанням одновимірного масиву:

тип [ ] [ ] назва = new тип [розмір] [розмір];

Як бачимо, після типу ми написали чотири квадратні дужки. Це ми створили оголоення до двовимірного масиву. Двовимірний масив - це таблиця.
Перші квадратні дужки - це довжина масиву, але довжина рядку

Другі квадратні дужки - це стовпчики.

Створимо приклад:

int [ ] [ ] numbers = new int [3] [5];

Цю команду можна розписати на два рядки:

int [ ] [ ] numbers;
numbers = new int [3] [5];

Отже, командою int та чотирма дужками разом зі змінною numbers оголошується змінна numbers двовимірного масиму. Внаслідок виконання команди

numbers = new int [3] [5];

Створюється двовимірний масив з цілих чисел: з трьох рядків і п'яти стовпців, а отже цей масив може містити 15 елементів. Щоб отримати елемент з масиву такого типу, потрібно записати наступний код:

System.out.println(numbers[1] [2]);

Ми отримаємо число з другого елемента рядка та який знаходиться у третьому ствопці. А тепер перейдімо до ініціалізації двовимірного масиву:

char [ ] [ ] s = {{'A', 'B', 'C'},{'D', 'E', 'F'}};
numbers = new int [3] [5];

Ось ми оголосили масив типу char та записали туди значення. Ще є певні дії з масивами:
.lenght - довжина масиву. Мається наувазі рядок, тобто для масиву розміром 3*3 довжина дорівнює 3 і для масиву 3*4 довжина дорівнює 3.
.clone - ця команда виділяє новий простір пам'яті і присвоює йому скопійований масив. Це певні команди, але не всі. Поки що досить.

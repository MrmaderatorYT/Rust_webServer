Оператор вибору (switch-case)

Оператор вибору, відомий як switch-case дозволяє виконувати різні блоки команд від значення деякого виразу. Оператор вибору розпочинається з ключового слова switch. Після цього ставляться дужки в яких ставиться вираз, який буде мати різні значення. Створімо простий приклад:

int x=2;
switch(x){

Ми написали тут вираз для перевірки значень в змінній x. Як бачимо ми тут відкрили дужки для перевірки значень. Щоб перевіряти значення, нам потрібне ключове слово case. Після ключового слова ми вказуємо значення, яке може набути. Ми порівнюємо зі значенням виразу в змінній x. Отже напишемо:

int x=2;
switch(x){
case 1:

Після значення, яке буде порівнюватися, ставиться двокрапка. Це так звані дужки, в яких буде виконуватися код. Давайте виведемо якесь значення в консoль:

int x=2;
switch(x){
case 1:
System.out.println("Число 1");

Після звершення опису коду контейнера для певного значення зазвичай ставиться ключове слово break
break зупиняє виконання команди, якщо отримаємо збіг перевірки. Створимо для інших значень:

int x=2;
switch(x){
case 1:
System.out.println("Число 1");
break;
case 2:
System.out.println("Число 2");
break;
case 3:
System.out.println("Число 3");
break; //Число 2

Але якщо нам потрібно вивести одне й те саме значення для різних даних, наприклад вивести в консоль: "Число 2" для значень 2 та 1. Ми можемо записати звичайним кодом, як ми робили, замінивши просто виконання коду, але для економії місця та часу можна зробити наступним чином:

int x=3;
switch(x){
case 1:
System.out.println("Число 1");
break;
case 2:
case 3:
case 4:
System.out.println("Число 2");
break;
Число 2

Але якщо в змінна набуде такого значення, якого не існує в блоці switch-case?
Для цього нам допоможе ключове слово default

default (з англ. за замовчуванням) - буде виконувати код в тому випадку, якщо значення змінної немає в контейнерах. Це ключове слово ставиться в самому кінці:

int x=100;
switch(x){
case 1:
System.out.println("Число 1");
break;
case 2:
System.out.println("Число 2");
break;
case 3:
System.out.println("Число 3");
break;
default:
System.out.println("Число N");
break;
Число N

Ще є певні варіанти написання цих блоків, але для пізніших версій, але це стандартна умова.

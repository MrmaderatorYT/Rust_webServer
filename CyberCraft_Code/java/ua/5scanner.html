Введення даних в консоль. Клас Scanner

Навіщо я виділив якийсь клас Scanner під цілий урок?
Scanner - клас, який допоможе вводити дані в консоль.

Початок роботи

Щоб почати працювати з цим класом, потрібно його спочатку імпортувати:

import java.util.Scanner;

імпорт відбувається завдяки ключовому слову import
Імпорт записується перед оголошенням класу:

import java.util.Scanner;
public class Main{
public static void main(String[] args){
}
}

Створімо тепер змінну типу Scanner:

Scanner scanner = new Scanner(System.in);

Scanner - імпортований клас, це як тип змінної, як int, але класу Scanner.
потім ми створили назву scanner назва може бути будь-якою.
Ми поставили символ "=", щоб присвоїти певні данні.
new потрібно для створення нового об'єкту на основі класу Scanner
Ми знову записали Scanner, щоб звернутися до методу класу Scanner.
System.in - параметр, який дозволить зчитувати дані з консолі, але цього не достатньо.
Ми повинні створити змінну:

String s = scanner.nextLine();

Як бачимо ми створили змінну s типу String та присвоїли значення змінної scanner, яку ми ствворили нещодавно.
.nextLine() - звертається до данних, знаходить там наступний рядок, який він ще не зчитував та повертає його.
Також є різні методи типу next:

  • .nextInt - зчитує і повертає введене число.

    • Замість nextInt можна вказати інший тип, як .nextByte і так далі

  • .hasNextInt - метод перевіряє, чи є наступна порція введених данних числом чи ні (повертає відповідно, true чи false

    • Замість hasNextInt можна вказати інший тип, як .hasNextByte і так далі

for-each цикл

For-each цикл - різновидність цикла for, котра використовується, коли потрібно опрацювати всі елементи масиву, або чогось іншого. Навіть переклад цього циклу (for each) означає з англійської - для кожного. Але слово each не використовується. Ось вам найпростіший варіант написання for-each:

for(type variable : list){
//код
}

type - тип змінної, яким повинен мати однаковий з типом елементів масиву.

variable - ім'я змінної.

list - список даних, які потрібно опрацювати.
Це був приклад при якому створюється нова змінна зі своїм типом. Ось варіант зі наявною змінною, яка була створена перед циклом:

int variable;
for(variable : list){
//код
}

Цей код можна записати звичайним циклом for:

int variable;
for(variable < list.length; variable++){
//код
}

Але використовуючи цикл for-each, будьте обережні з деякими обмеженнями, бо цей цикл менш універсальний за звичайний for

  • Якщо ви хочеите пройтись по списку з кінця, то використовуюяи цикл for можна так:
    for(int i = list.length-1; i>=0;i--){
    System.out.println(list[i]);
    }
    Але використовуючи for-each цього зробити неможливо.

  • for-each цикл не підходить, якщо потрібно внести зміни в масив, наприклад додати елемент, або змінити розмір масиву\

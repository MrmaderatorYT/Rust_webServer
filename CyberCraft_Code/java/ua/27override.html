Перевизначення методів

Буває таке, що в класі існує чимало методів з однаковою назвою, але різними аргументами. Уявімо у нас є один метод, який повинен виконувати різні речі, залежно від того, в якому класі він був виконаний:

public class Car{
public void Car1(){
System.out.println("Car");
}
}

Це батьківський клас і створимо інші в окремих файлах: One, Two

public class One extends Car{
@Override
public void car1(){
System.out.println("перша");
}
}

Та клас Two:

public class Two extends Car{
@Override
public void car1(){
System.out.println("друга");
}
}

І тепер в головному методі, класу Main ми викличемо класи, створивши об'єкт:

public class Main{
public static void main(String[] args){
Car car1 = new One();
Car car2 = new Two();
car1.car1();
car2.car1();
}
}

Перейдімо до нюансів:

  • Тип до перевизначеного методу не може відрізнятися від оригінального
  • Модифікатор доступу також має бути однаковим
  • У перевизначеного метода мають бути такі самі аргументи, як і в головного метода

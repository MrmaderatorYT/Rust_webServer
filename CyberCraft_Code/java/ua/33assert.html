Assert

Assert - це спеціальна конструкція, що дозволяє перевіряти припущення про значення довільних даних у довільному місці програми.
Ця конструкція може автоматично сигналізувати при виявлені некоректних даних, що зазвичай призводять до аварійного завершення програми. Але з першого вигляду - це дивна конструкція, яка спроможна "завалити" програму в невідповідний момент. І виходить питання "А навіщо вона нам?" Ця конструкція допоможе, якщо в якийсь момент часу деякі дані програми стали некоректними і ми не "завалили" програму відразу, а продовжили роботу, як ні в чому не бувало. Так як assert завершує програму відразу ж після виявлення некоректних даних, він дозволяє швидко локалізувати та виправити баги у програмі, що призвели до некоректних даних. Це його основне призначення.
Assert дозволяє відловлювати помилками у програмі на етапі компіляції або під час виконання. Синтаксиз конструкції простий:

assert умова;

Умова - вираз, який при обчислені повинен дати булевий результат. І якщо результат дорівнює true, то твердження є істиним і ніяких дій не виконується.
Якщо ж умова дає false, це означає, що стався збій і створюється об'єкт виключення за замовчуванням: AssertionError
Є ще інша форма написання assert:

assert умова : значення;

Значення - те, що передається конструктору AssertionError. Це значення перетворюється на рядок і відображається, якщо умова хибна. Ось вам приклад використання assert

public class Main{
static int val = 3;
static int getNum(){
return val--;
}
public static void main(String[] args){
int n;
for(int = 0; i<0; i++){
n=getNum();
assert n>0; //винекне збій, якщо n == 0
System.out.println("n дорівнює: " + n);
}
}
}
---------------------------------------
n дорівнює: 3
n дорівнює: 2
n дорівнює: 1
Exception in thread "main"...

Якщо ми запишемо іншим варіантом assert:

public class Main{
static int val = 3;
static int getnum(){
return val--;
}
public static void main(String[] args){
int n;
for(int i = 0; i<0; i++){
n=getnum();
assert n>0 : "n не підходить!"; //винекне збій, якщо n == 0
System.out.println("n дорівнює: " + n);
}
}
}
---------------------------------------
n дорівнює: 3
n дорівнює: 2
n дорівнює: 1
Exception in thread "main"...

if(n<0{
System.out.println(n менше нуля");
return;

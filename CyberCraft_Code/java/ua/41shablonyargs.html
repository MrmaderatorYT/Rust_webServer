Шаблони аргументів в узагальнених методах

Допустимo, що в нас є метод, який приймає дані якогось типу:

public int method(){
}

Але ніхто нам не забороняв вказувати невідомий тип. Шаблон аргументів вказується символом ? і це є невідомим типом:

boolean method(Class<?> name){
//описання
}

Також ми можемо обмежити діапазон об'єктів, вказавши суперклас:

public void method(ArrayList<? extends SuperClss> variable){
//описання
}

У цьому випадку можна використовувати класи, які є спадкоємниками SuperClss, а String та інші ми не можемо використовувати, якщо вони не використовувалися в SuperClss
